Setting up your Development Environment for Windows Phone

  1. Installing Windows 8 To develop for Windows Phone 8 you need to have Windows 8 installed. Luckily, if you're a U of T student you can get a free licensce for Windows 8 from dreamspark. If you're not from U of T your university may have a similar offer for you too. Create an account on Dreamspark then go to the "More Software through your school" tab to find Windows 8. U of T engineering link for Win8 (use ecf e-mail) Note: if you have Windows 7 installed you can develop for Windows Phone 7.5 apps but not Windows Phone 8 apps.
  2. Installing Visual Studio Development for Windows Phone or Windows 8 apps is done through Visual Studio. Students are once again able to get a free copy of Visual Studio Professional 2012 from Dreamspark.
  3. Installing the Windows Phone SDK You can install the Windows Phone SDK from here. This will install everything you need to build Windows Phone apps and will also let you install an express copy of Visual Studio if you have skipped step 2.
  4. Create Your First App Follow the "Create your first app Tutorial" from the windows phone dev site to familiarize yourself with the Visual Studio environment.
